# Contribution Guidelines

Thank you for your interest in `rolesetter`! When contributing, please first discuss the change you wish to make via issue.

This project has a [Code of Conduct](CODE-OF-CONDUCT.md). Please follow it in all your interactions with the project.

Contributions require signoff via the [Developer Certificate of Origin](https://developercertificate.org/) (DCO). A DCO-bot will check your PRs automatically.

## Before Submitting

Run all quality checks:

```shell
make pre
```

This runs formatting, linting (Go + YAML), unit tests, and vet in sequence.

## Code Reviews

All submissions, including those by project members, require review. We use GitHub pull requests for this purpose. Consult [GitHub Help](https://help.github.com/articles/about-pull-requests/) for more information on using pull requests.
